Core AURA

Main screen stays clean: the paper-aligned formula is central; details open on click.

TM–BM Resonance
STM/LTM explains where information lives. TM–BM resonance explains how meaning is generated: present context is interpreted through continuity-weighted memory history.
Bounded E₀
E₀ = tanh(R - D + λ_value). The tanh layer keeps output in (-1,+1), preventing unbounded meaning-state jumps.
λ_value Stabiliser
λ_value = clip(k·max(0,−R),0,λmax). It activates only under negative resonance and is clipped to prevent excessive correction.
Trainable R
Optional exploratory layer: BM weights update through feedback. Validated result: mean R improved, but wrong-anchor selection did not improve.
